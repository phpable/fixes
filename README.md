## Introduction
The bunch of functions extending the PHP standard library.

## Requirements
* PHP >= 7.2.0

## Install
There's a simple way to install ```able/fixes``` into your project via [Composer](http://getcomposer.org):

```bash
composer require able/fixes
```

## The list of function
* preg_count
